# Лабиринт сокровищ

Текстовая приключенческая игра на Python с загадками, предметами и событиями.

## Описание

Игра представляет собой текстовый квест, где игрок исследует лабиринт из нескольких комнат, решает загадки, собирает предметы и находит сокровища. Игра реализована полностью на Python с использованием только стандартной библиотеки `math` для генерации псевдослучайных событий.

## Установка

1. Клонируйте репозиторий:
git clone https://github.com/username/project1_Masenkova_Olesya_M25-555.git
cd project1_Masenkova_Olesya_M25-555


2. Установите зависимости через Poetry:
poetry install

## Запуск

Запустите игру командой:
poetry run project

Или через make:
make project

## Как играть

### Доступные команды:

- `north`, `south`, `east`, `west` — перемещение в указанном направлении
- `go <направление>` — альтернативный способ перемещения
- `look` или `l` — осмотреть текущую комнату
- `take <предмет>` — поднять предмет из комнаты
- `inventory` или `inv` — показать инвентарь
- `solve` — решить загадку в комнате или открыть сундук
- `help` — показать список команд
- `quit` или `exit` — выйти из игры

### Цель игры:

Найти ключ от комнаты с сокровищами, пройти через лабиринт, решить загадки и открыть сундук с сокровищем.
